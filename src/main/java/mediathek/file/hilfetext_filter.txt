Beim Filter wird zuerst mit der Angabe
"Zeitraum" eine verkürzte Liste erstellt.
Auf diese Liste werden dann die anderen
Filter angewandt. Das Arbeiten kann also
beschleunigt werden, wenn man den Zeitraum der
Filmliste einschränkt.

Es müssen der "Sender" und das "Thema" exakt
passen. Beim "Titel" muss die Eingabe im
Filmtitel enthalten sein.
Beim Feld "Thema/Titel" muss der Filter im
"Thema" ODER "Titel" enthalten sein. 
Im Filter "Irgendwo" wird zum Titel und Thema
auch die Beschreibung geprüft.

Ein Filterwort kann auch Leerzeichen enthalten,
mehrere Filterwörter werden durch "," getrennt,
z.B.: "Hallo ich,Tatort". Es wird dann nach Filmen
gesucht, die im entsprechenden Feld 
"Hallo ich" ODER "Tatort" enthalten.

Groß- und Kleinschreibung wird beim Filtern
nicht beachtet.

In den Filtern kann auch mit regulären Ausdrücken
gesucht werden. Diese müssen mit "#:" eingeleitet
werden und werden dann auf die entsprechenden
Felder des Films angewandt.

#:Abend.*
Das bedeutet z.B.: Es werden alle Filme gefunden, die 
im jeweiligen Feld mit "Abend" beginnen.

Auch bei den regulären Ausdrücken spielt Groß- und 
Kleinschreibung keine Rolle.

https://de.wikipedia.org/wiki/Regul%C3%A4rer_Ausdruck


Filterprofile:
==================
Mit einem Rechtsklick auf ein Profil kann der aktuell
eingestellte Filter gespeichert werden. Ein Klick
auf das entsprechende Profil, stellt dann den Filter
wieder so her.
Mit einem Rechtsklick kann man ein Profil auch löschen
oder die Anzahl der angezeigten Profile ändern.
